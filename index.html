Web Development I - Labo 2

Uitgewerkte oplossingen voor opdrachten 2.6 t.e.m. 2.19.

  • 2.6 Voorbeelden
    <p>Dit is een eenvoudige paragraaf</p>
    dit is een link
    <p>Dit is een paragraaf <a href='https://www.example.com'>met een hyperlink</a> en tekst</p>
  • 2.7 file:// vs http://
    file:// opent lokaal bestand zonder server
    http://localhost gebruikt een server en ondersteunt cookies en requests
  • 2.8 Whitespace
    Extra spaties of lege regels hebben geen invloed
    Gebruik &nbsp; voor extra spaties en <br> voor een nieuwe regel
  • 2.9 Entities
    Gebruik &amp;nbsp; en &lt;br /&gt; niet voor lay-out — daarvoor gebruiken we CSS
  • 2.10 Correct genest
    Enkel (4x(2+2)) is correct genest — andere missen haakjes of sluiten verkeerd af
  • 2.11 Mix van haakjes
    • ({}[]) - correct
    • {{{()] - fout
    • ({()}) - correct
    • (){[}] - fout
    • {{([]) - fout
  • 2.12 Foutieve nesting
    Fout: <strong> opent in <span> maar sluit na <a>
    Correct: <span>hier komt een <strong><a href=''>link</a></strong> te staan</span>
  • 2.13 Headings
    <h1>Titel 1</h1>
    <h2>Titel 2</h2>
    <h3>Titel 3</h3>
    Afbeelding voorbeeld
  • 2.14 Lijstfout
    <ol> moet binnen <li> staan
    Correct: <ul><li>twee<ol><li>jan</li></ol></li></ul>
  • 2.15 Website vs Webapplicatie
    Website = informatief
    Webapplicatie = interactief met data-opslag
    Gevoelige data hoort op de server
  • 2.18 Microsoft
    Taalinstelling wordt opgeslagen in cookies of localStorage (bekijk via Chrome Developer Tools)
  • 2.19 LG & Amazon
    LG gebruikt localStorage voor selectie
    Amazon bewaart winkelwagen server-side en koppelt via cookie-ID
